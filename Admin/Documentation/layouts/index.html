5. Layouts

Pacificonis template comes with some layouts for you to use.

5.1 Default

The default layout provide all crusial features for creating your app.

<!DOCTYPE html>
<html lang="en">

<head>

  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="msapplication-tap-highlight" content="no" />
  
  
  <!-- Chrome, Firefox OS and Opera -->
  <meta name="theme-color" content="#49CEFF">
  <!-- Windows Phone -->
  <meta name="msapplication-navbutton-color" content="#49CEFF" />
  <!-- iOS Safari -->
  <meta name="apple-mobile-web-app-capable" content="yes">
  <meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">

  <title>Default Layout</title>
  
  /* stylesheets */

    <!--[if lt IE 9]>
      <script src="bower_components/html5shiv/dist/html5shiv.min,js"></script>
      <script src="bower_components/respondJs/dest/respond.min.js"></script>
    <![endif]-->
</head>

<body>
  
/* preloader */

<div class="wrapper">
 <nav class="navbar">
   .....
 </nav>
 <aside class="sidebar">
   .....
 </aside>

  <div class="container-fluid">
    <div class="row">
      <!-- Content goes here-->
    </div>
  </div>
  <footer class="page-footer">.....</footer>
</div>


/* javascripts */

</body>
</html>

5.7 Three Column Layout

To fix both menu sidebar and right sidepanel add additional script to bottom of your page.

<body>
  
/* preloader */

<div class="wrapper">
 <nav class="navbar">
   .....
 </nav>
 <aside class="sidebar">
   .....
 </aside>

  <div class="container-fluid">
    <div class="row">
      <!-- Content goes here-->
    </div>
  </div>
  <footer class="page-footer">.....</footer>
</div>


/* javascripts */

<script>
    if($(window).width() >= 1440){
      $(".side-panel").addClass("open");
      $(".sidepanel-toggle").parent().addClass("open");
    }
    else{
      $(".side-panel").removeClass("open");
      $(".sidepanel-toggle").parent().removeClass("open");
      $("body").removeClass("three-column-layout");
    }
    
    $(window).resize(function(){
      if($(window).width() >= 1440){
        $(".side-panel").addClass("open");
        $(".sidepanel-toggle").parent().addClass("open");
        $("body").addClass("three-column-layout");
      }
      else{
        $(".side-panel").removeClass("open");
        $(".sidepanel-toggle").parent().removeClass("open");
        $("body").removeClass("three-column-layout");
      }
    });
</script>

</body>

5.8 Animations

Transitions in different animations are enabled by default. To disable transitions on basic sections and blocks add no-animations class to html tag.

5.9 Custom Scroll

Pacificonis provides custom scroll for template. To overwrite system`s default scroll add custom-scroll class to html tag.
