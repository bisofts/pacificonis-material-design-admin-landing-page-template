4. Template Structure

The general template structure is the same throughout the template. Here is the general structure.

Header

Main header is within a nav with an class name of navbar:

<nav class="navbar">
  <div class="navbar-header">
    ......
  </div>
  <div class="navbar-container clearfix">
    ......
  </div>
</nav>

Content

Content is within the main content area which is nested within a div with an classname of container-fluid:

<div class="container-fluid">
  <div class="row">
	<!-- Content goes here-->
  </div>
</div>

Site Menu

The site menu is within a aside with an classname of sidebar:

<aside class="sidebar">
  <ul class="nav metismenu">
    <li class="profile-sidebar-container">
      .....
    </li>
    <li>
      <a href="#"><i class="zmdi zmdi-view-dashboard"></i>Dashboard<span class="zmdi arrow"></span></a>
      <ul class="nav nav-inside collapse">
        <li class="inside-title">Dashboard</li>
        <li><a href="index.html">Dashboard v1</a></li>
        .....
      </ul>
    </li>
  </ul>
</aside>

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

  .class-name a {
    color: #somecolor;
  }

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.
