2. Building Tools

2.1 Install Node.js

First, you must download and install node.js (which includes npm). npm stands for node packaged modules and is a way to manage development dependencies through Node.js. You can check it in your terminal window using these commands node --version and npm --version. You can download Node.js here.

2.2 Working with Gulp

Gulp is awesome JavaScript task runner. If you are familiar with gulp, you can use it instead of Grunt.

Installing Gulp

From the command line:

  • Install gulp globally with npm install --global gulp.
  • Navigate to the root /pacificonis/ directory.
  • Then run npm install. npm will look at the package.json file and automatically install the necessary local dependencies listed there.

When completed, you'll be able to run the various Gulp commands provided from the command line.

Gulp Commands

gulp

You can modify or add your task in gulpfile.js.

Useful commands list:

  • gulp html: compile all html files.
  • gulp js: complied js file.
  • gulp scss: complied sass files.
  • gulp img: will copy all images and place them to build folder.
  • gulp browser-sync: started local webserver.
  • gulp watch: checking changes in files and fires particular task.

You can learn more about how to write gulp task files from http://gulpjs.com/
